Introducing the Channel Metadata API
Useful stats about your channels, including channel lifecycle state changes, occupancy changes or a list of active channels
View blog post
The new Channel Metadata API gives you access to insights related to your channels. Use the realtime library or Reactor to subscribe to channel lifecycle state-change events or channel occupancy updates. Alternatively, the REST subset of this API, called the Channel Status API, enables you to request the status of a channel or to list all the active channels within an app.
This API offers a flexible way to be notified when there are channel state or occupancy changes. This is especially useful if you want to, for example, keep track of the number of subscribers on a channel without using the presence feature, or to optimize what you publish to the Ably system by only publishing on channels where subscribers exist.
